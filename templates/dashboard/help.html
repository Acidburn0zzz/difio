Applications

In Difio you can import full set of application dependencies. This means telling Difio all package names and versions that you have currently deployed to run your application.

Difio will automatically monitor all packages on your account and generate analytics when new version is available. Application dashboard gives you access to all available analytics starting with the package versions you have imported. For example:

  • You import an application containing the package foo-1.0.1;
  • foo-1.0.2 has already been released previously. Analysis between foo-1.0.1 and foo-1.0.2 will be displayed for your application;
  • After a week foo-1.0.3 is released. Another analysis is generated between foo-1.0.1 and foo-1.0.3.

Your dashboard uses text statuses and color codes to identify applications which need update and/or include older packages!
